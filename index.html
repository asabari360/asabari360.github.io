Research focus

This page is written to match how teams actually describe the work: inflammation-first, protein evidence-heavy, and built for clarity.

Immune proteins & inflammation

We organize protein-level evidence around inflammatory responses—highlighting which proteins shift, which relationships tighten or break, and how that story evolves across conditions.

  • Protein changes (abundance / significance)
  • Interaction-centric summaries (PPI/graphs)
  • Clear figures for proposals and progress reports

Proteomics-friendly outputs

Proteomics generates a lot of detail fast. Our visuals are designed to keep the signal visible: volcano plots, heatmaps, ranked lists, and pathways-style panels.

  • Volcano + MA-style views
  • Heatmaps with clean labeling
  • Exportable PNG/SVG-ready layouts

Network biology that people can use

Network graphs are powerful, but only if they’re readable. We focus on interactivity (hover, filter, highlight) so collaborators can explore without a steep learning curve.

  • Filter by pathway / score / category
  • Show neighborhoods around a target protein
  • Explainability: “why is this node important?”
Note: This is a starter site. The demos use small synthetic data so the page is fast. You can swap in real datasets later (CSV/JSON) without changing the layout.

Platform roadmap (what we can add next)

Start simple, then grow into a real research portal as you connect datasets, papers, and projects.

Short-term add-ons

  • Upload a CSV → render volcano + heatmap automatically
  • “Target protein” search box → show neighborhood and top partners
  • Download figures (PNG/SVG) and tables (CSV)
  • Project pages for grants (aims, figures, milestones)

Long-term direction

  • Interactive dashboards per study (conditions, tissues, cohorts)
  • Pathway overlays + literature-linked evidence
  • Versioned datasets and reproducible figure generation
  • Private access for collaborators (if you move off GitHub Pages)
